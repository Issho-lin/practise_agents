# 语音指令协议对接说明

## 1. 顶层返回结构
### 1.1 方案1：结构化输出

```json
{
  "rawText": "用户原始话术",
  "intentType": "command",
  "chatReply": "",
  "confidence": 0.0,
  "actions": []
}
```

- `rawText`：原文透传，不改写
- `intentType`：意图分类（`command | chat | unknown`）
- `chatReply`：当 `intentType=chat` 时返回给前端的自然语言回复
- `confidence`：0~1
- `actions`：有序动作列表

### 1.2 方案2: 流式输出

- 文本正常流式输出
- 结构化片段用固定标签包裹：`<json>...</json>`
```
好的，我来帮你切到对标看板并筛选 2026 年半年度数据……
<json>{"intentType":"command","actions":[{"type":"navigate","pageId":"opcenter","targetCode":"dui_biao_kan_ban","rawValue":"去对标看板"},{"type":"setBoardType","value":3,"rawValue":"对标看板"},{"type":"setStartYearPeriod","value":"2026","rawValue":"2026年"},{"type":"setEndYearPeriod","value":"半年度","rawValue":"半年度"}]}</json>
已为你处理，正在加载页面数据。

```

## 1.3 顶层字段说明

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `rawText` | `string` | 是 | 用户原始输入文本，用于日志、回溯、澄清上下文 | 不改写、不清洗 |
| `intentType` | `string` | 是 | 意图分类，决定前端走“执行链路”还是“对话链路” | 枚举：`command` / `chat` / `unknown` |
| `chatReply` | `string` | 条件必填 | 当 `intentType=chat` 时，后端模型返回给前端的自然语言回复 | `intentType=chat` 必填且建议非空；其它类型不传或空字符串 |
| `confidence` | `number` | 是 | 后端解析置信度，表示本次动作解析可信程度 | 范围 `0~1`；当 `intentType=unknown` 建议置 `0` |
| `actions` | `CommandAction[]` | 是 | 当前语音指令拆解出的有序动作序列 | `intentType=command` 时建议非空；`chat/unknown` 时应为空数组 |

## 1.4 Action 通用字段说明

| 字段 | 类型 | 必填 | 含义 | 备注 |
|---|---|---|---|---|
| `type` | `string` | 是 | 动作类型标识，决定前端执行分支 | 如 `navigate`/`setMonth`/`clarify` |
| `rawValue` | `string` | 否 | 类似于提取出来的动作和实体名称，如`打开规模看板` | 用于日志与用户反馈，不参与主校验 |

## 1.3 `intentType` 路由语义

| `intentType` | 含义 | 前端处理方式 | `actions` 约束 | `chatReply` 约束 |
|---|---|---|---|---|
| `command` | 可执行业务指令 | 进入 Dispatcher/Orchestrator 执行 | 建议非空；有 `navigate` 时必须在首位 | 不传或空字符串 |
| `chat` | 闲聊/问答类输入 | 不执行页面动作，走对话回复链路 | 必须为空数组 `[]` | 必填且建议非空 |
| `unknown` | 无法稳定识别 | 不执行页面动作，走兜底提示 | 必须为空数组 `[]` | 可选（可返回兜底话术） |

> 建议：对话回复统一放在 `chatReply`，避免前端再调用一次聊天接口。

## 1.5 各动作专属字段说明

### `navigate`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `pageId` | `string` | 是 | 目标页面逻辑 ID | 当前为 `home` 或 `opcenter` |
| `targetCode` | `string` | 是 | 目标页面路由/功能编码 | 需与前端菜单编码一致 |

### `fullscreen`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `action` | `string` | 是 | 全屏控制动作 | `enter` 或 `exit` |

### `setBoardType`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `value` | `number` | 是 | 看板类型值 | `1/2/3` |

### `setOrg`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `value` | `string` | 是 | 组织名称 | 必须命中前端注入 `orgs` 白名单 |

### `setYear`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `value` | `string` | 是 | 年份值 | 格式 `yyyy`，示例 `"2026"` |

### `setMonth`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `value` | `number` | 是 | 月份值 | 整数 `1..12` |

### `setDay`（预留）

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `value` | `number` | 是 | 日期值 | 整数 `1..31`；页面不支持时建议不下发 |

### `setStartYearPeriod`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `value` | `string` | 是 | 对标周期起始条件中的年份部分 | 格式 `yyyy` |

### `setEndYearPeriod`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `value` | `string` | 是 | 对标周期结束条件中的周期类型部分 | 枚举 `年度` 或 `半年度` |

### `setPageTab`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `value` | `string` | 是 | 页面级 Tab 名称 | 必须命中前端注入 `tabs` 白名单 |

### `openModule`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `moduleId` | `string` | 是 | 目标模块唯一标识 | 需与前端注册模块 ID 一致 |

### `setModuleTab`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `moduleId` | `string` | 是 | 目标模块唯一标识 | 需与前端注册模块 ID 一致 |
| `controlId` | `string` | 是 | 模块内控件唯一标识 | 需与前端注册控件 ID 一致 |
| `value` | `string\|number` | 是 | 目标选项值 | 对应控件 options 中的 value |

### `setSelect`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `moduleId` | `string` | 是 | 目标模块唯一标识 | 需与前端注册模块 ID 一致 |
| `controlId` | `string` | 是 | 模块内条件控件标识，预留字段，以防以后一个模块有多个同类型的控件 | 需与前端注册控件 ID 一致：compareTab、trackSelect|
| `value` | `string\|number` | 是 | 目标选项值 | 对应控件 options 中的 value |

### `clarify`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `reason` | `string` | 是 | 告知用户为何需要澄清 | 用户可读文案 |
| `moduleIds` | `string[]` | 是 | 需要用户二次确认的模块候选 ID 列表 | 非空数组 |

### `noop`

| 字段 | 类型 | 必填 | 含义 | 约束 |
|---|---|---|---|---|
| `reason` | `string` | 是 | 该动作不执行的原因说明 | 用户可读文案 |

## 2. 动作类型与取值约束

## 2.1 导航类

### `navigate`

```json
{ "type": "navigate", "pageId": "home|opcenter", "targetCode": "xxx", "rawValue": "..." }
```

- 有 `navigate` 时，必须出现在 `actions[0]`

### `fullscreen`

```json
{ "type": "fullscreen", "action": "enter|exit", "rawValue": "..." }
```

---

## 2.2 页面筛选类

### `setBoardType`

```json
{ "type": "setBoardType", "value": 1, "rawValue": "整体看板" }
```

- 取值：`1 | 2 | 3`

### `setOrg`

```json
{ "type": "setOrg", "value": "华南区域公司", "rawValue": "华南区域公司" }
```

- 取值：必须在前端注入 `orgs` 列表

### `setYear`

```json
{ "type": "setYear", "value": "2026", "rawValue": "2026年" }
```

- 取值格式：`yyyy`

### `setMonth`

```json
{ "type": "setMonth", "value": 3, "rawValue": "3月" }
```

- 取值：整数 `1..12`

### `setDay`（预留）

```json
{ "type": "setDay", "value": 15, "rawValue": "15日" }
```

- 取值：整数 `1..31`
- 页面不支持日级筛选时，建议后端不下发

### `setStartYearPeriod`（对标看板）

```json
{ "type": "setStartYearPeriod", "value": "2026", "rawValue": "2026年" }
```

- 取值格式：`yyyy`

### `setEndYearPeriod`（对标看板）

```json
{ "type": "setEndYearPeriod", "value": "半年度", "rawValue": "半年度" }
```

- 取值枚举：`年度 | 半年度`

### `setPageTab`

```json
{ "type": "setPageTab", "value": "区域公司（榜单）", "rawValue": "区域榜单" }
```

- 取值：必须在前端注入 `tabs` 选项

---

## 2.3 模块控制类

### `openModule`

```json
{ "type": "openModule", "moduleId": "opcenter:group-revenue", "rawValue": "集团营收" }
```

- 适用场景：模块内部没有可切换筛选条件，仅支持命中后直接聚焦放大展示
- 执行语义：前端根据 `moduleId` 打开对应模块详情弹窗，底层映射到已实现的 `openModal` 能力
- 取值：`moduleId` 必须命中前端模块注册表；若未命中，前端应返回模块未找到反馈

### `setModuleTab`

```json
{ "type": "setModuleTab", "moduleId": "opcenter:branch-revenue", "controlId": "compareTab", "value": 1, "rawValue": "按历史占比" }
```

### `setSelect`

```json
{ "type": "setSelect", "moduleId": "mktdash:branch-new-contract", "controlId": "trackSelect", "value": "住宅", "rawValue": "住宅赛道" }
```

---

## 2.4 特殊动作

### `clarify`

```json
{ "type": "clarify", "reason": "存在多个候选模块，请确认", "moduleIds": ["a", "b"], "rawValue": "趋势图" }
```

### `noop`

```json
{ "type": "noop", "reason": "该模块当前不可执行语音切换", "rawValue": "..." }
```

## 3. 覆盖场景示例

## 3.1 无法识别

```json
{
  "rawText": "今天天气不错",
  "intentType": "unknown",
  "confidence": 0,
  "actions": []
}
```

## 3.2 全屏

```json
{
  "rawText": "打开全屏",
  "confidence": 0.95,
  "actions": [
    { "type": "fullscreen", "action": "enter", "rawValue": "打开全屏" }
  ]
}
```

## 3.3 仅导航

```json
{
  "rawText": "打开业绩预警看板",
  "confidence": 0.9,
  "actions": [
    { "type": "navigate", "pageId": "home", "targetCode": "ye_ji_yu_jing", "rawValue": "业绩预警看板" }
  ]
}
```

## 3.4 导航 + 对标周期

```json
{
  "rawText": "去对标看板看2026年半年度数据",
  "intentType": "command",
  "confidence": 0.92,
  "actions": [
    { "type": "navigate", "pageId": "opcenter", "targetCode": "dui_biao_kan_ban", "rawValue": "去对标看板" },
    { "type": "setBoardType", "value": 3, "rawValue": "对标看板" },
    { "type": "setStartYearPeriod", "value": "2026", "rawValue": "2026年" },
    { "type": "setEndYearPeriod", "value": "半年度", "rawValue": "半年度" }
  ]
}
```

## 3.5 组织 + 年月筛选

```json
{
  "rawText": "看华南区域2026年3月数据",
  "intentType": "command",
  "confidence": 0.88,
  "actions": [
    { "type": "setOrg", "value": "华南区域", "rawValue": "华南区域" },
    { "type": "setYear", "value": "2026", "rawValue": "2026年" },
    { "type": "setMonth", "value": 3, "rawValue": "3月" }
  ]
}
```

## 3.6 日级筛选（仅支持页面下发）

```json
{
  "rawText": "看华南区域2026年3月15日数据",
  "confidence": 0.9,
  "actions": [
    { "type": "setOrg", "value": "华南区域", "rawValue": "华南区域" },
    { "type": "setYear", "value": "2026", "rawValue": "2026年" },
    { "type": "setMonth", "value": 3, "rawValue": "3月" },
    { "type": "setDay", "value": 15, "rawValue": "15日" }
  ]
}
```

## 3.7 模块歧义澄清

```json
{
  "rawText": "切到趋势图",
  "intentType": "command",
  "confidence": 0.72,
  "actions": [
    {
      "type": "clarify",
      "reason": "发现多个模块包含“趋势图”选项，请确认具体模块",
      "moduleIds": ["area_trend", "biz_dist_trend"],
      "rawValue": "趋势图"
    }
  ]
}
```

## 3.8 模块聚焦放大（无内部筛选条件）

```json
{
  "rawText": "打开集团营收详情",
  "intentType": "command",
  "confidence": 0.84,
  "actions": [
    { "type": "openModule", "moduleId": "opcenter:group-revenue", "rawValue": "集团营收" }
  ]
}
```

> 说明：适用于命中后仅需放大查看详情的模块，不包含模块内 tab/select 等二级筛选动作。

## 3.9 模块控制（tab + select）

```json
{
  "rawText": "切换下属新增签约到按区域并选住宅赛道",
  "intentType": "command",
  "confidence": 0.86,
  "actions": [
    { "type": "setModuleTab", "moduleId": "mktdash:branch-new-contract", "controlId": "compareTab", "value": 1, "rawValue": "按区域" },
    { "type": "setSelect", "moduleId": "mktdash:branch-new-contract", "controlId": "trackSelect", "value": "住宅", "rawValue": "住宅赛道" }
  ]
}
```

## 3.10 命中不可执行模块

```json
{
  "rawText": "把集团当年可转化营收切到按贡献占比",
  "intentType": "command",
  "confidence": 0.8,
  "actions": [
    { "type": "noop", "reason": "该模块当前不可执行语音切换", "rawValue": "集团当年可转化营收按贡献占比" }
  ]
}
```

## 3.11 闲聊（不执行动作）

```json
{
  "rawText": "今天天气不错",
  "intentType": "chat",
  "confidence": 0.93,
  "actions": []
}
```
